# The Rupture Axes — an anti-convergence framework

LLM-generated design converges: dark canvas, lime or purple accent, Space
Grotesk / Plus Jakarta / DM Sans, vertical scroll, three equal feature
columns. Every generated site drifts toward the same site.

The fix is structural, not cosmetic: before designing anything, commit to
**one axis of rupture** — one deliberate break from the default — and build
the whole kit around it. One axis per kit. Never blend two.

## The eight axes

### 1. Broken navigation
The site does not scroll vertically page-after-page. Rooms, panels,
horizontal drift — the user *travels* instead of scrolling.
*Published example: [Atlas Drift](https://www.stacklift.design/kit/atlas-drift).*

### 2. Structural motion
A drag, inertia, or momentum physics is the interface — not a decoration on
top of it. Remove the motion and the site stops working.
*Published examples: [Glass Meridian](https://www.stacklift.design/kit/glass-meridian) (horizontal drag), [Slipstream](https://www.stacklift.design/kit/slipstream) (vertical drag).*

### 3. Off-grid layout
Collage, diagonals, overlap, deliberate misalignment. The grid exists only
to be visibly escaped.
*Published examples: [Loomhall](https://www.stacklift.design/kit/loomhall) (mosaic), [Paste Theory](https://www.stacklift.design/kit/paste-theory) (collage).*

### 4. Extreme density
Either maximalist saturation or radical emptiness — both refuse the
comfortable middle.
*Published examples: [Riot Pop](https://www.stacklift.design/kit/riot-pop) (maximalist), [Terra Firma](https://www.stacklift.design/kit/terra-firma) (ultra-minimal, scroll-as-dial).*

### 5. Progressive revelation
Typography or photography that *builds itself* as you scroll — the gesture
is the content appearing.
*Published examples: [Marginalia](https://www.stacklift.design/kit/marginalia) (typographic highlight), [Emulsion](https://www.stacklift.design/kit/emulsion) (photo development).*

### 6. Cursor as protagonist
The cursor stops being a pointer and becomes an instrument — a reticle, a
lens, a magnet. The scene answers it.
*Published example: [Gauss](https://www.stacklift.design/kit/gauss) (magnetic inspection reticle).*

### 7. Spatial choreography
One traveling element crosses the whole site; scrolling reorganises the
scene around it.
*Published example: [Deadwax](https://www.stacklift.design/kit/deadwax) (traveling vinyl disc).*

### 8. Dimensional depth
Real perceived depth — layered parallax planes, CSS 3D — as the structure of
the page, not an effect on a card.
*Published example: [Sightline](https://www.stacklift.design/kit/sightline) (scroll-travel museum enfilade).*

## How to use the framework

1. Ask: which axis has this project *not* seen yet?
2. Commit to one. Write it at the top of your working notes.
3. Every design decision afterwards either serves the axis or is a rest zone.
4. At review: if the axis is not obvious in 5 seconds of use, it is too weak.
