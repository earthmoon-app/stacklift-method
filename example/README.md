# Example — Solar Draft

A complete kit built with this method, exactly as published in the
[Stacklift catalogue](https://www.stacklift.design/kit/solar-draft) (free tier).

- **Axis:** pre-framework kit — a clean warm-agency baseline showing the
  production standard (self-contained HTML, design.md, Design System view).
  It predates the method's `axis:` front-matter field and attribution line,
  so don't expect either in this kit's `design.md`.
- **Open it:** double-click `solar-draft/index.html`. Everything is inline —
  no build step, no server.
- **Read `solar-draft/design.md`** to see the format the method produces —
  the file an AI agent uses to adapt the kit.

Live demo: <https://www.stacklift.design/kit/solar-draft>
